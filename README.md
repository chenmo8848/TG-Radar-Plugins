# TG-Radar-Plugins

External plugin repository for TG-Radar.

## Layout

- `plugins/admin/` — Admin command plugins
- `plugins/core/` — Core runtime plugins

## Included plugins

- `general.py`
- `folders.py`
- `rules.py`
- `routes.py`
- `system.py`
- `keyword_monitor.py`

Deploy this repo next to the core repo:

- `/root/TG-Radar`
- `/root/TG-Radar-Plugins`
